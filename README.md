## 🚀 About This Project
A Python-based business management system for stone trading companies. Calculate profits, track deals, and manage your stone empire like a tycoon.

- **Empire Rank System**: Track your growth from Pebble to Mountain
- **Data Visualization**: Charts with matplotlib + pandas
